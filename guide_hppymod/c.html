Cara Menginstal HappyMod

Untuk menginstal aplikasi HappyMod kita perlu melakukan secara manual. Caranya mudah, cukup ikuti langkah-langkah di bawah ini dengan benar.

  1. Buka Pengaturan Android dan tekan Keamanan ( atau Privasi, tergantung pada versi firmware Android ponsel )
  2. Temukan opsi Aplikasi Tidak Dikenal dan tekan untuk mengaktifkannya
  3. Buka aplikasi file manager dan temukan lokasi folder Unduhan.
  4. Temukan apk HappyMod dan tekan 2x file .apk nya
  5. Ikuti instruksi di layar dan instalasinya akan dimulai
  6. Ketika ikon HappyMod muncul di layar beranda, artinya sudah siap digunakan
  7. Selesai
